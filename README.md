# AI学伴 · 义务教育信息平等与资源匹配平台

面向义务教育阶段（小学、初中）学生的轻量化学习辅助工具。帮助学生每日记录学习内容与疑惑，由 AI 自动分析并推荐免费公开学习资源与学习计划，重点解决乡村学生、学困生"无人辅导、不会找资源"的痛点。

---

## 功能特性

- **个人信息注册**：首次使用填写昵称、年级、学校、地区，信息本地保存，无需注册账号
- **每日学习记录**：选择科目（语文/数学/英语等9科），填写"今天学了什么"和"哪里不懂"
- **AI 智能分析**：保存记录后自动调用 Kimi AI，返回知识点总结、免费学习资源链接、个人学习计划
- **学习中心**：查看所有历史记录及对应的 AI 分析结果，支持按日期展开/折叠
- **学习统计**：累计记录天数、学科次数、提问数、连续打卡天数
- **数据本地化**：所有记录存储在浏览器 `localStorage`，无需服务器数据库

## 快速启动

### 1. 安装依赖

```bash
pip install openai flask flask-cors
```

### 2. 配置 API Key

在 `server.py` 第 19 行填入你的 Kimi API Key：

```python
client = OpenAI(
    api_key="sk-你的key",
    base_url="https://api.moonshot.cn/v1",
)
```

或使用环境变量：

```bash
export KIMI_API_KEY="sk-你的key"
```

### 3. 启动后端

```bash
python server.py
```

默认监听 `http://localhost:5002`，启动成功后终端会显示：

```
AI学伴后端启动中，端口 5002
 * Running on http://127.0.0.1:5002
```

### 4. 打开前端

直接用浏览器打开 `index.html` 文件即可，无需额外配置。

## 使用流程

1. 首次打开 `index.html`，填写个人信息（昵称、年级、学校）
2. 每天上线后，点击科目标签选择今日学习的科目
3. 在对应科目下填写"今天学了什么"和"哪里不懂"
4. 点击"保存今日记录"，页面自动调用 AI 分析并展示结果
5. 点击导航栏"我的中心"可查看所有历史记录和 AI 分析

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | 纯 HTML / CSS / JavaScript，无框架依赖 |
| 数据存储 | 浏览器 `localStorage` |
| 后端 | Python + Flask |
| AI 模型 | Kimi `moonshot-v1-8k`（月之暗面） |
| 跨域 | Flask-CORS |

---

## 注意事项

- 后端仅在调用 AI 分析时需要运行，浏览记录、编辑个人信息等操作无需后端
- 清空浏览器数据会导致本地记录丢失，如需长期保存建议定期导出
- 本项目为公益性质，请勿用于商业用途
